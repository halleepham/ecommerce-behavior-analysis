## recommendations.parquet

One row per user per recommendation. Contains the top-K category recommendations generated by the category-based collaborative filtering model for each user in the training set. Ranked by predicted affinity score. Primary input for time-sensitive re-ranking (08).

| Column | Description |
|---|---|
| `user_id` | Foreign key to `user_category_affinity` |
| `recommended_category` | Top-level product category recommended for this user |
| `predicted_score` | Weighted sum of the user's affinity scores multiplied by the category-category similarity scores. Higher = stronger predicted affinity. |
| `rank` | Rank of this recommendation for this user (1 = highest predicted score). Each user has up to 5 recommendations. |

**How to load:**

```python
recommendations = pd.read_parquet(
    '/kaggle/input/datasets/halleepham/collaborative-filtering-outputs/recommendations.parquet'
)
print(recommendations.shape)  # (11138954, 4)
```

## loo_test.parquet

One row per eligible user. Contains the leave-one-out test set used for evaluation in notebooks 06 and 08. For each eligible user (those with 2+ categories after applying the minimum affinity threshold of 5), the highest affinity category was held out as the test item.

Users with only one category after filtering are excluded — holding out their one category would leave no training signal for the model.

| Column | Description |
|---|---|
| `user_id` | Foreign key to `user_category_affinity` |
| `held_out_category` | The category held out as the test item — the user's highest affinity category at the time of the split |

**How to load:**

```python
loo_test = pd.read_parquet(
    '/kaggle/input/datasets/halleepham/collaborative-filtering-outputs/loo_test.parquet'
)
print(loo_test.shape)  # (567192, 2)
```

## How these files were built

The recommendations were generated using a category-based item-based collaborative filtering model. A minimum affinity threshold of 5 was applied to filter weak interactions. A 13×13 category-category cosine similarity matrix was computed from the filtered user-category interaction matrix, then adjusted using shrinkage weighting (factor=50) to penalize category pairs with few shared users. Predicted affinity scores were computed for each user's unseen categories using a weighted sum of their known affinity scores and the category-category similarity scores. The top 5 categories by predicted score were returned as recommendations.

The leave-one-out test set was constructed by holding out each eligible user's highest affinity category before building the training interaction matrix.

See `notebooks/06_collaborative_filtering.ipynb` for the full derivation logic.
